This accessibility statement applies to www.furleypage.co.uk
We aim to give the best experience to as many users as possible. Below is a list of what you will be able to do on our website, followed by some detailed instructions. Zoom in up to 400% without the text spilling off the screen (this will increase the size of the font and images)
Change the font size using your browser (Chrome, Firefox, Safari, Edge)
Change colours and contrast levels using your browser (Chrome, Firefox, Safari, Edge)
Navigate the website using just the keyboard
Interact with the website using screen reading software (including recent versions of JAWS, NVDA and VoiceOver) We also try to make the website text as simple as possible to understand.
If you have a disability, we have some instructions below for how to increase the accessibility on your device. If you can’t find what you are after AbilityNet has a comprehensive guide for changes you can make to different devices/ browsers.
Zoom into a website to increase font and image size On desktop devices you can zoom in by pressing ctrl and + (cmd and + on a Mac)
Alternatively, you can press ctrl and move the mouse wheel up (on Window devices) to zoom in Changing the font size (desktop)
Chrome Click the menu button ( in the top-right hand corner of Chrome)
Click ‘Settings’
In the left-hand sidebar click ‘Appearance’
Look for ‘Font size’ and then adjust to your preference Firefox Click the menu button ( in top-right hand corner of Firefox)
Click ‘Settings’ in the dropdown menu
Under ‘Fonts and colours’ you can change the font and size to your preference Safari Click ‘Safari’ in the top-left hand corner
Click ‘Preferences’
Click ‘Advanced’
Tick ‘Never use font smaller than’ and select your preference Edge Click the menu button ( in the top-right hand corner)
Click ‘Appearance’
Look for ‘Font size’ and then adjust to your preference Change colours and contrast levels
Chrome You can install a Chrome Plugin called High Contrast Firefox Click the menu button ( in top-right hand corner of Firefox)
Click ‘Settings’ in the dropdown menu
Under ‘Fonts and colours’ click on ‘Colours’
Select your preferred colours
Click the dropdown under ‘Override the colours specified by the page with your selections above’
Select ‘Always’ Safari
Safari doesn’t have any built-in options for changing the default colour settings. Though if you go to the apple logo ( top-left hand corner) Click ‘System preferences’
Click ‘Accessibility’
Click ‘Display’
Then there is a tick box for ‘Increase contrast’, as well as other options you can try Edge You can install an Edge Plugin called High Contrast Navigate the website using just your keyboard
If you have problems using a mouse, then you can move through the website page using just the tab button. If you want to go backward you can press tab while holding down shift What to do if you cannot access parts of this website
If you need information on this website in a different format, then contact us and we will be happy to help. Report accessibility problems with this website
We are always looking to improve the accessibility of our website. Contact us if you have any issues with the accessibility of our website or if you have any suggestions. BACK TO TOP Careers
